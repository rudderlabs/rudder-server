TL;DR

Adds a label-gated GitHub Actions pipeline that runs four Codex stages to generate and publish a PR review site to GitHub Pages.

  • 🧭Introduces .github/workflows/ai-preview.yaml to build, deploy, and tear down per-PR previews on gh-pages.
  • 🧩Replaces one-shot prompting with a four-stage pipeline driven by .github/codex/agents/{analyze,diagrams,risks,assemble}.md.

Why

The PR reduces reviewer ramp-up time on unfamiliar changes by producing one visual review page from PR diff and repo context, with opt-in label gating and teardown on PR close.

Risks

medium

Deploy commit message includes only PR number, so retries or regenerated previews can produce identical commit messages and make git push origin gh-pages fail as a non-fast-forward when concurrent updates land.

Mitigation: Include a unique value (for example PR head SHA or run ID) in the gh-pages commit message or pull/rebase before push in .github/workflows/ai-preview.yaml.

medium

workflow_dispatch is declared but the job-level if requires github.event.pull_request.*, so manual runs are effectively dead and can fail expression evaluation or always skip unexpectedly.

Mitigation: Either remove workflow_dispatch or add explicit dispatch-safe guards/inputs so manual execution has defined behavior in .github/workflows/ai-preview.yaml.

medium

The teardown path resets branch history with git checkout --orphan gh-pages and force-pushes when no gh-pages branch is detected, which can wipe unrelated content if branch detection is wrong or transiently unavailable.

Mitigation: Avoid orphan force-push for recovery paths, or require an explicit safeguard check that existing remote content is empty/expected before destructive branch recreation.

low

The pipeline now depends on four stage prompts under .github/codex/agents/; prompt drift or malformed stage output can block preview generation even when code changes are valid.

Mitigation: Keep strict schema validation (already present) and add lightweight prompt contract tests on representative fixture artifacts to catch breakage early.
